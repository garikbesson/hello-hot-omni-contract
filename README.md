# hello-hot-omni-contract

The example of a near smart contract that interacts with Hot Omni smart contract (depositing, checking balances, withdrawing, and transferring functionality).

## How to Build Locally?

Install [`cargo-near`](https://github.com/near/cargo-near) and run:

```bash
cargo near build
```

## How to Test Locally?

```bash
cargo test
```

## How to Deploy?

To deploy manually, install [`cargo-near`](https://github.com/near/cargo-near) and run:

```bash
cargo near deploy build-reproducible-wasm <account-id>
```
